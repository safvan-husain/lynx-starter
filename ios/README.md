# iOS Host App

The Swift iOS host app is available at:

- `ios/lynx-ios/`

Quick start:

1. `cd ios/lynx-ios`
2. `pod install`
3. Open `lynx-ios.xcworkspace`
4. Run the app in the simulator

The app loads the Lynx bundle from:

- `http://127.0.0.1:3000/main.lynx.bundle`

Start the Lynx dev server from `lynx/` before launching the host app:

- `pnpm run dev`
